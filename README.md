# Proyecto App Salon - MEVN

## Descripción

Este proyecto es una aplicación de gestión de salones de belleza que permite a los usuarios realizar reservas, administrar citas, y gestionar los servicios ofrecidos por el salón.

## Tecnologías Utilizadas

- **Backend:**
  - Node.js
  - Express
  - MongoDB

- **Frontend:**
  - Vue.js
  - Pinia
  - TailwindCSS
  - Vue Router

## Características

### Backend

- API RESTful para gestionar usuarios, servicios, citas, etc.
- Conexión a una base de datos MongoDB para almacenar y recuperar datos.

### Frontend

- Interfaz de usuario interactiva y moderna.
- Integración de Vue Router para la navegación entre páginas.
- Utilización de Pinia para el manejo del estado de la aplicación.
- Diseño responsivo con TailwindCSS.

## Instalación

**Clonar el repositorio:**

   ```bash
   git clone https://github.com/tu-usuario/app-salon-mevn.git


Instalar las dependencias del backend:

cd backend
npm install express


Instalar las dependencias del frontend:

cd frontend
npm install


Uso

Iniciar el servidor backend:

cd backend
npm run dev


Iniciar la aplicación frontend:

cd frontend
npm run dev

Abrir tu navegador web y navegar a http://localhost:5173 para ver la aplicación en funcionamiento.
